CSS Position

FIRST RULE OF POSITION: DO NOT ABUSE POSITION

The position property defines how an element is poisition on your page.

there are five values for the position property.

  1. Static
  2. Realtive
  3. Absolute
  4. Fixed
  5. Sticky

Static

The default value for positition. The element is position normally according to your page flow

Static Position

Relative

The element is position realtive to it's normal position.

Relative Position

top 50px, left 100px

Absolute

The element is positioned relative to its nearest postioned ancestor

If you have an element absolutey positioned, you should relatively position its parent.

Absolute Position

Fixed

The element is positioned relative to the viewport.

Position Fixed

Stick

The element toggles between relative and fixed, depending on scroll position.
